# Degen Cookie - 演示指南 🍪

## 🚀 快速開始 (5分鐘)

### 1. 啟動前端
```bash
cd cookie-frontend
npm run dev
# 打開: http://localhost:5173
```

### 2. 演示流程

#### **步驟 1: 連接錢包** (30秒)
- 點擊 "Connect" 按鈕
- 選擇 IOTA 錢包
- 批准連接
- ✅ 看到錢包地址顯示

#### **步驟 2: 領取空投** (30秒)
- 看到 "餘額: 0.00 CKIE"
- 點擊 "🎁 領取 100 CKIE 空投"
- 在錢包中簽署交易
- ✅ 餘額更新為 100 CKIE

#### **步驟 3: 玩遊戲 - 贏** (15秒)
- 輸入賭注: 1 CKIE
- 點擊 "開始挑戰"
- **立即**點擊餅乾 (在 ~3-4秒時)
- ✅ 看到倍數 (1.1x - 1.4x)
- ✅ 餘額增加到 ~101-104 CKIE

#### **步驟 4: 玩遊戲 - 輸** (15秒)
- 輸入賭注: 1 CKIE
- 點擊 "開始挑戰"
- **等待** (不點擊，讓它在 3-8秒爆炸)
- 💥 餅乾爆炸！
- ✅ 餘額減少 1 CKIE

## 🎯 演示重點

### **什麼是 Degen Cookie？**
> "一個在 IOTA 區塊鏈上的爆炸式博弈遊戲，使用自定義 CKIE 代幣。餅乾會在 3-8 秒內成長。提早收官 = 小利潤。等越久 = 越大利潤但有爆炸風險！"

### **技術亮點**
- ✅ **自定義代幣**: CKIE 具有鑄造/銷毀機制
- ✅ **智能合約**: 用 Move 編寫，部署在 IOTA 測試網
- ✅ **經濟模型**: 贏 = 鑄造新代幣，輸 = 永久銷毀代幣
- ✅ **可證明公平**: 使用 drand 隨機信標決定爆炸時間
- ✅ **時間窗口**: 3-8 秒，指數級獎勵 (最高 3.5倍 在 8秒)

## 🐛 故障排除

### 錢包無法連接？
- 確保已安裝 IOTA 錢包擴展
- 切換到測試網
- 刷新頁面

### 交易失敗？
- 檢查是否有測試網 IOTA 用於 gas
- 從此處獲取: https://faucet.testnet.iota.cafe/

### 餘額未更新？
- 等待 3-5 秒確認
- 需要時刷新頁面
